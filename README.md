Simulador de Inversiones

En esta aplicación los usuarios podrán comprar y vender activos financieros simulados utilizando un saldo virtual. Cada activo tendrá un precio que podrá variar con el tiempo.
Los usuarios podrán ver un listado de activos disponibles, consultar su valor actual y operar sobre ellos. Además, cada usuario tendrá un perfil donde podrá ver su saldo, los activos que posee y un historial de sus operaciones.
Las operaciones se realizarán de forma directa al precio actual, sin simular un mercado complejo. Habrá dos tipos de usuario: inversor y administrador.
